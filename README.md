# VPSM
A worse version of [HCloud](https://hetzner.cloud) for now.
![VPSM Logo - A Beaver with a server on his back](docs/images/Logo.jpg)

## Usage
```
vpsm create --name my-server --image ubuntu-22.04 --type cx11 --ssh-key ~/.ssh/id_rsa.pub
```
